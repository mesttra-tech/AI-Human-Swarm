# Publication Checklist

Before publishing content to this repository, confirm:

- No live `.env` values, API keys, tokens, private keys, cookies, or OAuth material.
- No customer data, private logs, private operational notes, internal URLs, or local machine paths.
- No offensive exploit chains, weaponized payloads, persistence mechanisms, stealth instructions, or obfuscated code.
- No commands that print environment variables, auth files, cookies, SSH keys, shell history, or secret stores.
- No shortened links, opaque downloads, or `curl | bash` style instructions.
- Current-threat claims include public sources.
- AI-assisted content is disclosed in the pull request.
- Runnable commands are defensive, reviewable, and include verification steps.
- Daily patches are generic public guidance, not private scan results.
- Maintainer review is complete before merge.
